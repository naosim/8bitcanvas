# 開発原則

AIへの戒めとして遵守すること。
- 返答は日本語にすること
  - 返答文言を一度見返し、中国語がないか確認すること
  - 特に次の単語は中国語になりがちなので、変換する
    - 进 → 進
    - 种类 → 種類
    - 生 线 → 生の線
    - 曲线的选择 → 曲線の選択
    - 无限 → 無限
    - 坐标 → 座標
    - 基准点 → 基準点
    - 风格 → 風格
    - 足的 → 十分
- 矛盾や欠陥を見つけたらすぐに指摘する
- 質問ではなく問題提起として伝える
- 未解決問題は文書化して残す
- 「メンテせよ」と言ったら SPEC.md, RULES.md, HISTORY.md, AGENTS.md を更新する
- Markdown では太文字`**`を使わない
- 制作工程：要件定義→設計→実装の順に進めます。設計完了までにSPEC.mdを完成させます。その後、SPEC.mdをインプットに実装をします。
