## 要件
- 無限の広さを持つキャンバスアプリ
- UIが8bitでかわいい
- 出力ファイル形式はJSON Canvas (Obsidian Canvas形式)
- 機能
  - JSON Canvasを編集できる基本的な機能
  - undo/redo
  - ズーム
- 技術
  - vanilla HTML / CSS / JavaScript
  - 保存先はローカルファイル(DL)、LocalStorage

## 設計

### 対応フォーマット
Obsidian Canvas形式
- ノード：テキスト、角丸矩形、円
- エッジ：矢印（始点/終点）

### UIデザイン
- ピクセルサイズ：16x16
- キャンバスサイズ：256x256（論理ピクセル）
- 配色：ファミコン風（限られたパレット）
- フォント：DotGothic16（Webフォント）
- メニュー配置：下部
- ツールバー：アクションボタン配置（テキスト、矩形、円など）
- プロパティバー：選択項目の編集フォーム配置

### キャンバス仕様
- 原点：画面の中心 (0, 0)
- 無限キャンバス：パンとズームで移動可能

### 基本機能

#### ノード作成・編集・削除
- テキストノード：ダブルクリックでテキスト編集
- 矩形ノード：ダブルクリックでテキスト編集（矩形内にテキスト表示）
- 円ノード：テキスト挿入不要、直径14px
- テキスト配置：横方向（左揃え/中央揃え/右揃え）、縦方向（上揃え/中央揃え/下揃え）
- 矩形サイズ自動調整：テキストがはみ出す場合、矩形を自動拡大

#### カラー指定
- **24色カラーパレット**：背景色・線色共にパレットから選択
- **パレット編集**：
  - パレットを**ダブルクリック** → ブラウザ標準のカラーピッカーが開く
  - カラーピッカーで色を選択 → パレットの色が更新される
  - パレット色は全ノード共通（1つのパレットを編集すると、その色をを使っている全ノードに影響）

#### テキストノードの特殊機能
- **背景透明化**：チェックボックスで背景を透明にできる
- **線透明化**：チェックボックスで枠線を透明にできる
- 矩形・円ノードには透明オプションなし（常に表示される）

#### エッジ作成・削除
- 作成方法：SHIFT押しながら2つのノードを選択→エッジボタンクリック
- エッジ編集：選択したエッジのプロパティで「始点矢印/終点矢印」を切り替え
- エッジは矩形の角から出る
- エッジは常に矩形の前面に表示

#### 選択・移動
- 単一選択：クリック
- 複数選択：SHIFT+クリック
- 矩形選択中にTabキーで新規矩形作成＆接続

#### その他
- undo/redo
- ズームイン/アウト
- ファイル保存（JSON Canvas形式）
- ファイル読み込み
- JSONをコンソールに出力（Logボタン）
- 選択項目の重なり順序変更（最前面/最背面）。エッジよりは後ろ
